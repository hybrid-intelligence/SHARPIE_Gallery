# Amaze

A lightweight maze navigation task generator for sighted AI agents.

## Installation

From the gallery root:
```bash
python install.py amaze
```

## Dependencies

- amaze-benchmarker[all]

## Configuration

This use case has the following agents:

- **Agent** (agent_0): human inputs (policy: Amaze)
  - Keyboard controls:
    - ↑ (Good)
    - ↓ (Bad)

See `config.json` for full configuration details.
