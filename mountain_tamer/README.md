# Mountain car TAMER

This is a classic experiment by Knox & Stone in which an agent is trained with evaluative feedback.

## Installation

From the gallery root:
```bash
python install.py mountain_tamer
```

## Dependencies

- gymnasium[classic_control]

## Configuration

This use case has the following agents:

- **Mountain car TAMER agent** (agent_mct): human inputs (policy: Mountain car TAMER)
  - Keyboard controls:
    - ↑ (Good)
    - ↓ (Bad)

See `config.yaml` for full configuration details.
